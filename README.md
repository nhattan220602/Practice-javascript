## Front-end Practice Projects

This repository showcases practical front-end projects focused on interactive UI development, data-driven components, and clean user experience execution.

## Projects

### 1) Dashboard-design
An analytics dashboard prototype with:
- KPI cards that update by time period
- Revenue trend visualization (canvas bar chart)
- Theme toggle with persisted user preference

### 2) Music-webpage
A browser-based music player featuring:
- Playlist-based playback controls (play/pause, previous, next)
- Seek bar and live duration tracking
- Volume control and keyboard shortcuts

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript (ES6)
